## BCS Network Device Monitoring Service

This is a simple web-app that will register a device, receive status updates and displaying their current operational state..

Stack used:

- ReactJs
- TypeScript

How to run:
- Clone the repo ([here](https://github.com/nelsonkimaiga/networkDeviceMonitoringServiceClientSide.git))
- open the rootsfolder on your terminal
- run npm install from root of the folder
run `npm run dev` and the application will run on port 5173 in your browser i.e: http://localhost:5173/
